## ğŸ†˜ Troubleshooting

### "Authentication failed"

**Solution 1: Use Personal Access Token**
1. Go to: https://github.com/settings/tokens
2. Generate new token (classic)
3. Select scopes: `repo`, `workflow`
4. Copy the token
5. When prompted for password, paste the token

**Solution 2: SSH Key**
```bash
# Change remote to SSH
git remote set-url origin git@github.com:mpichadze-bot/observability-agent-tutorial.git
git push -u origin main
```

### "Repository doesn't exist"

Create it first:
1. Go to: https://github.com/organizations/mpichadze-bot/repositories/new
2. Name: `observability-agent-tutorial`
3. Click "Create repository"
4. Then run `git push -u origin main`

### "Permission denied"

Make sure you have access to the `mpichadze-bot` organization:
- Check: https://github.com/mpichadze-bot
- You need to be a member or owner
